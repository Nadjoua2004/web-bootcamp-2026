# Quiz Sounds

The following sound files have been downloaded to this directory (`public/sounds/`):

1.  `boo.mp3`
    - Played when the score is 0-39.
2.  `ok.mp3`
    - Played when the score is 40-69.
3.  `clap.mp3`
    - Played when the score is 70-100.

Default paths used in `ScoreDisplay.jsx`:
- `/sounds/boo.mp3`
- `/sounds/ok.mp3`
- `/sounds/clap.mp3`
